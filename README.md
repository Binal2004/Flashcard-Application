FlashWise is a modern digital flashcard application designed to help students learn, revise, and test concepts efficiently.
The application provides organized flashcards, quiz-based learning, and a clean, user-friendly interface to improve study productivity.

Features
1.Flashcard-Based Learning
Learn concepts through interactive flashcards
2.Quiz & Self-Assessment
Test knowledge using MCQs and instant result evaluation
3.Search Functionality
Quickly find flashcards by subject or topic
4.Subject-Wise Organization
Flashcards grouped by topics such as ASP.NET, C++, etc.
5.User Profile Management
View and manage personal learning progress
6.Modern UI Design
Clean layouts with smooth navigation
7.Responsive Design
Works smoothly across devices
